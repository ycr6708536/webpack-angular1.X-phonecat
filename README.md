# webpack-angular1.X-phonecat

> 使用 webpack 打包 angular 1.X 的例子，打包了PhoneCat 项目 
###安装

项目地址：（`git clone`）

```shell
git clone https://github.com/ycr6708536/webpack-angular1.X-phonecat.git
```

## Build Setup

``` bash
# 通过`npm`安装本地服务第三方依赖模块
npm install

# 启动服务(http://localhost:8090)
npm run start

# 发布代码
npm run build


```
